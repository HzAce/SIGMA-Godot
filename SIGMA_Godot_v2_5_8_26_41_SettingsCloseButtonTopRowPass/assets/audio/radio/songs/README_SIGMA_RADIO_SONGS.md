# SIGMA Radio Song Pack

This folder contains the runtime OGG Vorbis versions of the user-uploaded DJ SIGMA WAV masters.

Runtime rule:
- Long music and SIGMA Radio tracks use OGG Vorbis.
- WAV masters stay outside the runtime project.
- Short SFX may remain WAV.

Menu override rule:
- Only one SIGMA Radio song can be assigned as Main Menu music at a time.
- Off-Air only removes a song from SIGMA Radio rotation.
- Off-Air assigned Menu Music still plays as Main Menu music when SIGMA Radio is OFF.

Suggested menu music track:
- sigma_radio_04_sum_menu_theme
