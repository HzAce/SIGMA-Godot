# SIGMA Godot Build AI Context v1.1.21

Build: Pass-and-Play Comfort + Turn Handoff.

Current rule direction is unchanged:
- Guardian moves, jump-captures, Deploys, and checks Surround orthogonally.
- Monarch Peril is direct threat.
- Monarchs Surrender instead of being captured or removed.
- Friendly boxed-in non-Monarch pieces Retreat.
- Enemy boxed-in non-Monarch pieces are removed and count as enemy removals.
- Advanced Cap = max 3 of each advanced type on board.
- Retreated advanced pieces return to advanced supply and stop counting against the on-board cap while off board.

This build adds local two-player comfort:
- Turn Handoff screen after completed turns.
- Start Turn button for the next player.
- Optional Board Flip setting, default OFF.
- Turn Handoff setting, default ON.
- Pass-and-play settings saved in user://sigma_user.cfg.

Do not add AI, online, monetization, Vanguard, or new rules in this build branch.
