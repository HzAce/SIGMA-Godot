# Classic SIGMA Board 3D Blockout

This is a simple OBJ prototype for the Classic SIGMA Board. It is not final AAA art; it is a production bridge for future Blender/Godot work.

Use cases:

- Collections board viewer prototype
- Scale/blockout reference for a future GLB model
- Material planning for the premium 2.5D gameplay board

The live Android gameplay board should remain 2.5D/readable first. A richer 3D model can be used in Collections later.
