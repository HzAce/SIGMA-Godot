# SIGMA v2.5.7 Android Export Path

Export debug APK to:

```text
exports/android/SIGMA_Mobile_v2_5_7_debug.apk
```

Recommended install command on macOS after export:

```bash
~/Library/Android/sdk/platform-tools/adb install -r "exports/android/SIGMA_Mobile_v2_5_7_debug.apk"
```

Release AAB placeholder:

```text
exports/android/SIGMA_Mobile_v2_5_7_release.aab
```

Do not commit private keystore files.
